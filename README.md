# Weather App

A responsive React weather dashboard that shows live current weather and a 5-day forecast for cities in Canada using the OpenWeather API.

## Features

- Search weather by city
- Current weather conditions
- 5-day forecast
- Celsius and Fahrenheit toggle
- Responsive layout for desktop and mobile
- Clean portfolio-ready UI

## Tech Stack

- React
- Vite
- Tailwind CSS
- OpenWeather API
